A# 📵 AppLocker — スマホ依存から抜け出すアプリ

> 「触る必要がない」と思わせることで、スマホ依存を根本から解決する iOS アプリ

---

## 🎯 コンセプト

既存のスクリーンタイム系アプリは「触るのがめんどくさい」と思わせるだけで、依存の根本を解決できていません。

**AppLocker** は、アプリを開こうとする瞬間に意識を介入させ、「本当に今開く必要があるか？」を問いかけることで、**無意識な使用習慣そのものを変える**ことを目指します。

---

## 🧩 背景・課題

スマホの過剰利用は、現代の若者に深刻な影響を与えています。

| カテゴリ | 具体的な問題 |
|---|---|
| 時間の浪費 | 逃げ道として使用・勉強中に触ってしまう |
| 心身への影響 | 集中力低下・睡眠不足・学力低下 |
| 社会的影響 | 対人関係の希薄化 |
| 精神的影響 | SNS 疲れ・うつ状態 |
| 既存アプリの限界 | 面倒さで抑制するだけで根本解決できていない |

---

## 💡 解決アプローチ

### 基本方針

- 「触る必要がない」と自分で気づかせる
- **無意識**な使用を**意識的**な選択に変える
- 時間よりも**回数**に着目し、依存度の変化を可視化する

### 認知行動療法モデル（ABCDE）の活用

ユーザーが自分の使用パターンを振り返るための思考フレームワークを実装します。

| ステップ | 内容 | 例 |
|---|---|---|
| **A**ctivating event | きっかけ | 携帯の通知が鳴った |
| **B**elief | 自動思考 | 「5分だけなら大丈夫」 |
| **C**onsequences | 結果 | ショート動画を30分浪費・罪悪感8/10 |
| **D**isputation | 反論 | 本当に今すぐ見る必要があった？ |
| **E**ffect | 新しい考え・行動 | 通知をオフにする・帰宅後はスマホをしまう |
